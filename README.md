# Kyan Health Predictive Audit Prototype

High-fidelity React prototype of the Kyan Health Predictive Audit tool.

## Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS (v4 via PostCSS)
- Framer Motion
- Lucide React

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Open in Browser**
    Visit `http://localhost:5173` (or the URL shown in your terminal).

## Screens
1.  **Search**: Enter "on.com" for the perfect match flow, or any other domain for the generic flow.
2.  **Loader**: Simulated enrichment process.
3.  **Confirmation**: Verify company details.
4.  **Vent**: Sentiment analysis simulation (triggers on inputs > 10 chars).
5.  **Sliders**: Adjust "Hidden Cost" proxies.
6.  **Iceberg**: Visual cost breakdown.
7.  **Gate**: Lead capture and mock booking.
